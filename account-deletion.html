Taptly Account Deletion

You can request deletion of your Taptly account and associated data at any time.

How to delete your account

  1. Open the Taptly mobile application
  2. Go to Profile
  3. Select "Delete account"
  4. Confirm deletion

Data that will be deleted

  • Name
  • Email address
  • Phone number
  • Social media usernames
  • Shared contact history

Data retention

All personal data is permanently deleted immediately after confirmation. No data is retained except minimal technical logs required for security and fraud prevention, which are automatically removed within 30 days.
